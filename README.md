# PDF 工具箱

一个基于 Python 的 PDF 处理工具，提供合并、拆分、旋转、排序、PDF 转图片、图片转 PDF 等常用功能。界面使用 Tkinter 构建，支持大文件懒加载，操作直观。

## 功能列表

- **合并 PDF**：将多个 PDF 文件按顺序合并为一个。
- **拆分/删除页面**：加载 PDF 后，可勾选要删除的页面，保存为新的 PDF。
- **PDF 转图片**：将 PDF 的指定页面导出为 PNG、JPG 或 JPEG 格式，支持设置 DPI。
- **图片转 PDF**：将多张图片（PNG、JPG、BMP）按顺序合并为一个 PDF。
- **页面旋转**：对选中的页面进行 90°、180°、270° 旋转，支持预览和重置。
- **页面排序**：拖动或移动页面顺序，重新排列 PDF 页面。

## 运行环境

- Python 3.8 或更高版本
- 支持 Windows / macOS / Linux（需安装对应依赖）

使用说明
通用操作
每个功能选项卡中都有 浏览 按钮，用于选择 PDF 文件或图片。

处理大文件时，缩略图采用懒加载，滚动到底部自动加载后续页面。

所有导出操作均需选择保存路径。

合并 PDF
点击“添加 PDF”添加文件，可在列表中调整顺序或移除文件。

点击“开始合并”选择输出路径即可。

拆分/删除页面
打开 PDF 后，缩略图默认显示所有页面，单击页面可多选。

点击“删除选中页面”并选择保存路径，即可生成不含选中页的新 PDF。

PDF 转图片
打开 PDF 后，默认全选所有页面，可手动勾选要导出的页面。

点击“导出页面”，设置图片格式（PNG/JPEG）和 DPI（72–600），选择输出文件夹。

图片转 PDF
添加图片后，可调整顺序、置顶/置底，点击“开始转换”保存为 PDF。

页面旋转
选中需要旋转的页面（支持多选），选择旋转角度，点击“预览旋转”会打开临时文件查看效果。

点击“保存旋转”将旋转结果保存为新 PDF。

页面排序
打开 PDF 后，缩略图按当前顺序排列，单击选中一个页面。

使用“上移”“下移”“置顶”“置底”调整顺序，最后点击“保存排序”输出新 PDF。

注意事项
处理超大 PDF（数百页以上）时，首次加载缩略图可能需要几秒钟，请耐心等待。

旋转功能中的“预览旋转”会生成临时 PDF 文件，存放在系统临时目录，不会自动删除，可手动清理。

输出文件请勿与原文件同名，以免覆盖。
